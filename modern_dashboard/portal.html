Cara Akses Dashboard

Via Tailscale (Remote Access):
  1. Install Tailscale di device Anda
  2. Login dengan akun yang di-share
  3. Pastikan status "Connected"
  4. Buka link Tailscale di browser

  2. Buka link localhost atau local IP
  3. Tidak perlu Tailscale
  4. Hanya bisa diakses dari jaringan lokal
